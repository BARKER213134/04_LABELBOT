# 🤖 Telegram Bot - Quick Start

## Тестовый Бот

**Username:** @whitelabel_shipping_bot_test_bot  
**Статус:** ✅ Активен и готов к использованию

### Как использовать:

1. **Откройте Telegram**
2. **Найдите:** `@whitelabel_shipping_bot_test_bot`
3. **Нажмите START** или отправьте `/start`

### Доступные команды:

- `/start` - Приветствие и инструкции
- `/create` - Создать shipping label (полный wizard)
- `/help` - Справка по командам
- `/cancel` - Отменить создание лейбла

### Процесс создания лейбла:

```
/create
  ↓
📍 Адрес отправителя (6 полей)
  • Имя
  • Улица, дом
  • Город
  • Штат (2 буквы: TX, CA, NY...)
  • ZIP код (5 цифр)
  • Телефон (можно skip)
  ↓
📍 Адрес получателя (6 полей)
  • Имя
  • Улица, дом
  • Город
  • Штат
  • ZIP код
  • Телефон (можно skip)
  ↓
📦 Параметры посылки
  • Вес в унциях (16 oz = 1 фунт)
  • Размеры: Длина Ширина Высота в дюймах
    Например: 12 8 6
  ↓
🚚 Выбор перевозчика (кнопки)
  [📦 USPS] [✈️ FedEx] [🚚 UPS]
  ↓
📋 Выбор сервиса (кнопки)
  Например для USPS:
  - Priority Mail
  - First Class Mail
  - Ground Advantage
  ↓
✅ Подтверждение
  Показывается полный summary
  [✅ Создать лейбл] [❌ Отменить]
  ↓
🎉 Готово!
  Tracking number и стоимость
```

### Важные заметки:

**Валидация:**
- Штат: ОБЯЗАТЕЛЬНО 2 буквы (TX, не Texas)
- ZIP: ОБЯЗАТЕЛЬНО 5 цифр
- Вес: Положительное число
- Размеры: 3 числа через пробел

**Опциональные поля:**
- Телефон можно пропустить, написав `skip`

**Отмена:**
- В любой момент отправьте `/cancel` чтобы остановить создание

### Troubleshooting:

**Бот не отвечает?**
```bash
# Проверьте webhook
cd /app/backend
python3 setup_dual_webhooks.py

# Перезапустите backend
sudo supervisorctl restart backend
```

**Ошибка при создании лейбла?**
- Проверьте что адреса валидны (US адреса)
- Убедитесь что все поля заполнены правильно
- Проверьте формат (штат - 2 буквы, ZIP - 5 цифр)

**Застряли в conversation?**
- Отправьте `/cancel`
- Или начните заново с `/create`

### Логи:

Для отладки:
```bash
# Backend логи
tail -f /var/log/supervisor/backend.out.log | grep telegram

# Ошибки
tail -f /var/log/supervisor/backend.err.log
```
